PicPics

Description

This project was developed as the main project of a web applications course. It utilizes the MERN stack, which is MongoDB, Express.js, React.js, and Node.js. The project is a social media platform with google-drive-esque accessibility. Logged in users can upload, download, delete, like, sort and rename images. The biggest challenge was definitley making sure that the entire program was fully tested, extensible and scalable. The project was a great learning experience and a great way to be introduced to the MERN stack, as well as typescript. During the process, I enjoyed both the frontend development in general, and the backend server's development and communication with the frontend.
