BombIT

Description

This prject was developed while learning object-oriented programming principles. It used Java, LibGDX engine as well as Gradle build and ended up being a 2-player game that can be played on the same keyboard. The biggest challenge was learning the best object-oriented method that delayed the explosion of bombs without exposing the entirety of the bomb interface for outer classes.
